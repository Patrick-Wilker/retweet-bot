<h1 align="center"> Retweet Bot </h1>

Um bot para retweetar periodiamente. Para rodar bastar dar um <strong>npm run start </strong> no terminal.

<h2> Tecnologia utilizada </h2>

* Node.js (JavaScript)

<h2> Ferramentas </h2>

* Visual Studio Code

<h2>Pacotes instalados</h2>

* npm init
* npm install –-save twit
